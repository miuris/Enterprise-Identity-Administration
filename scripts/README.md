# Microsoft Graph PowerShell Scripts

This directory contains reusable PowerShell automation scripts created during the Enterprise Identity Administration implementation.

## Included Scripts

- 01-Connect-MgGraph.ps1
- 02-Get-AllUsers.ps1
- 03-Get-AllGroups.ps1
- 04-Export-DirectoryReport.ps1
- 05-DisabledUsersReport.ps1
- 06-LicensedUsersReport.ps1

These scripts demonstrate administrative automation using the Microsoft Graph PowerShell SDK for Microsoft Entra ID.
