# Phase 1 – Tenant Administration


## 🎯 Objective

Deploy the Microsoft 365 tenant for Kimble Glass and establish the enterprise identity foundation.

---

## Business Requirement

Kimble Glass is migrating from an on-premises identity environment to Microsoft Entra ID.

The objective is to establish the cloud tenant before user provisioning begins.

---

⚙️ Implementation

- Verified tenant
- Configured administrator account
- Validated licensing
- Reviewed Microsoft 365 Admin Center
- Verified Microsoft Entra ID synchronization readiness

## 🤔 What Surprised Me

I was surprised to learn that a personal Microsoft account cannot administer Microsoft 365 services. Enterprise identity management requires an organizational tenant, appropriate licensing, and administrator permissions. Understanding this distinction helped me better appreciate how Microsoft separates consumer services from enterprise identity infrastructure.
